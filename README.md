Aprendizagem Django — Estrutura Básica

Este repositório foi criado com o objetivo de estudar e praticar os fundamentos do framework Django, com foco na compreensão da estrutura básica de um projeto, organização de pastas e funcionamento dos principais componentes do framework.

O projeto serve como base educacional para consolidar conhecimentos em backend com Python, seguindo boas práticas iniciais adotadas pela comunidade Django.

Objetivos do Projeto

Compreender a arquitetura básica do Django
Aprender a criar e organizar projetos e apps
Entender o fluxo de requisição e resposta (request/response)
Trabalhar com rotas, views e templates
Utilizar o ORM do Django
Aplicar migrações e gerenciamento do banco de dados
Executar o servidor de desenvolvimento
